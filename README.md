# Babel-Library-Book-Fetch
This python script fetches the selected book from the famous Online Babel Library (https://libraryofbabel.info/) (Don't know what the Babel Library is? https://en.wikipedia.org/wiki/The_Library_of_Babel) it lets the user input the hex, wall, shelf, and volume number of a book and it gives back the actual book

This program was only tested on Windows even though it should work on Linux and maybe MacOS

## About the program

## Why should I use this?

Because it is an easy, lightweight solution in case you need to download whole books from the babel library.
Plus, it adds an option to just store book IDs instead of whole books saving up a ton of space on long texts


## How to use

Open terminal and navigate to the directory where you have cloned/downloaded the repository
When you have reached the directory in Command Prompt type

```
python main.py
```

✅ It should work fine!

### Three options will appear:
1. Automatic mode
2. Manual mode
3. Convert Manual into Automatic query

#### Automatic mode:
Paste an Automatic query (generated with the third option) in the input field and press enter
The book should download automatically and store itself in a file named BabelBookOut.txt

#### Maual mode:
Paste the hex, wall, shelf, and volume
The book should download and store itself in a file named BabelBookOut.txt

#### Convert Manual into Automatic query mode:
Paste the hex, wall, shelf, and volume
The automatic query should display in a file named AutoQueryOut.txt, just open the file and copy the query ID
